# API格式

## 服务器地址

ws://localhost:2345

## 发送格式

```json
{
    "cmd": "Cal", //命令
    "id": "0x01", //从机ID
    "data": [参数],
}
```

## 接收格式

```json
{
    "id": "0x01", //从机ID
    "status": "success", //状态 success | error
    "data": [接收buffer转hex字符串],
}
```

## 繁忙回复

```json
{
    "status": "busy",
}
```

## 功能列表

### 1. Cal

用于校准闭环步进电机

cmd: "Cal"
data: []

### 2. Mv

用于移动滑台。

cmd: "Mv"
data: [脉冲数量]

参数栏只有一个参数，即步进电机的脉冲数量，绝对位置移动 0~120000 [0~15mm]。

3200 个脉冲，电机转一圈，移动4mm。

### 3. MvSetting

用于设置位置模式移动参数。

cmd: "MvSetting"
data: [加速度,速度]

- 加速度：范围 0~255。默认值为100。
- 速度：范围 0~5000rpm。默认值为1200。

### 4. Stop

用于停止移动。

cmd: "Stop"
data: []

### 5. Zero

用于滑台归零。

cmd: "Zero"
data: []

### 6. ZeroSetting

用于设置滑台归0参数。

cmd: "ZeroSetting"
data: [归零速度,回零超时时间,碰撞回零检测转速,碰撞回零检测电流,碰撞回零检测时间]

- 归零速度：范围 0~5000rpm。默认值为300。
- 回零超时时间：范围 0~999999ms。默认值为10000。
- 碰撞回零检测转速：范围 0~5000rpm。默认值为300。
- 碰撞回零检测电流：范围 0~6000mA。默认值为800。
- 碰撞回零检测时间：范围 0~65535ms。默认值为60。

### 7. ZeroStatus

用于查询回零状态。

cmd: "ZeroStatus"
data: []

这个需要解析返回数据，判断是否零位完成。

例如

```json
{
    "id": "0x01",
    "status": "success",
    "data": "0103020003f845"
}
```

需要看 01030200 [03] f845 方括号中的字段，03 表示回零完成，这个位置是固定的。
